## üí° Tips

1. **Icon Variants**: Most icons have numbered variants (01, 02, 03)
   - Example: `Arrow01Icon`, `Arrow02Icon`, `Arrow03Icon`

2. **Naming Pattern**: Icons follow PascalCase with "Icon" suffix
   - Format: `DescriptionNumberIcon`
   - Example: `ArrowLeft01Icon`

3. **Common Prefixes**:
   - `Circle` - Circular versions
   - `Square` - Square versions
   - `Outline` - Outlined versions
   - `Solid` - Filled versions

4. **Search Strategy**:
   - Think of the action/object first
   - Try numbered variants (01, 02, 03)
   - Check both singular and plural forms
